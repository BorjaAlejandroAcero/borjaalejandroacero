Para la realización del video se ha Utilizado SimpleScreenRecorder, descargable gratuitamente de los repositorios de Ubuntu. Un microfono externo ha sido necesario para que la calidad sea aceptable sin ser muy exigentes. El Ide con el que se ha escrito el código es SublimeText 3 que como podreis ver se puede usar sin necesidad de adquirirlo. Como navegador se ha usado Firefox en su versión más reciente para linux El formato de video elegido es webm que es compatible con casi todos los navegadores y ocupa bastante poco para una calidad mas que aceptable para un video de una pantalla llena de código
